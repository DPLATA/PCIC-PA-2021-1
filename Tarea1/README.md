# Instrucciones para correr los programas:

### Para correr el programa del árbol h:
* descargar el archivo
* correr ```python3 htree.py```

### Para compilar y correr el archivo de matrices:
* descargar el archivo
* compilarlo ```gcc matriz.c -o matriz```
* correrlo ```./matriz```
